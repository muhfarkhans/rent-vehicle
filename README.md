# Rent Vehicle

## How to run

1. dcomposer install
2. cp .env.example .env
3. php artisan key:generate
4. setup your database credentials
5. php artisan migrate
6. php artisan db:seed
7. php artisan serve
8. <http://localhost:8000/>
